# my-betterfox-setup

Configuration for firefox that uses the [betterfox](https://github.com/yokoffing/Betterfox) improvements

### Apply configuration

1. Open Firefox. In the URL bar, type `about:profiles` and press Enter.
2. For the profile you want to use, click Open Folder in the Root Directory section.
3. Move the user.js file into the folder.
